# 고도리 데스크톱 타이머

타이머가 작동하는 동안 고도리가 Windows 화면 우측 하단에서 노트북으로 함께 작업하는 Electron 앱입니다.

## 실행

```cmd
cd /d C:\work\godori-timer
npm start
```

웹 서버와 포트를 사용하지 않습니다.

## 동작

- `산책 시작`을 누르면 작은 고도리가 화면 우측 하단에 나타납니다.
- 투명 배경의 동화풍 고도리 캐릭터만 보이며 화이트·실버 오버이어 헤드폰을 쓰고 노트북으로 작업합니다.
- 고도리 캐릭터는 부자연스러운 합성 애니메이션 없이 정지 상태로 표시됩니다.
- 말풍선은 고도리가 화면에 표시되는 동안 계속 유지되며 상태에 따라 문구가 바뀝니다.
- 타이머가 끝나면 고도리 위에 완료 팝업이 유지됩니다.
- 사용자가 1~180분 사이의 시간을 직접 입력해 추가하거나 타이머를 종료할 수 있습니다.
- 고슴도치 고도리, 고양이 나비, 햄스터 몽실, 토끼 보리 중 함께할 캐릭터를 선택할 수 있습니다.
- 선택한 캐릭터는 즉시 데스크톱 위젯에 반영되고 다음 실행에도 유지됩니다.
- 고양이는 선글라스를 쓴 힙한 디자인이며, 토끼는 더 복슬복슬한 아기 토끼 디자인입니다.
- 캐릭터 아래 컨트롤 바에서 잔여시간을 확인하고 일시정지 또는 초기화할 수 있습니다.
- 고도리를 직접 드래그해 원하는 위치에 놓을 수 있고 다음 실행에도 그 위치가 유지됩니다.
- 작업표시줄 아래나 화면 밖으로 놓으면 가장 가까운 보이는 위치로 자동 보정됩니다.
- 일시정지하면 조용히 쉬고, 초기화하면 화면에서 사라집니다.
- 해상도나 작업표시줄 위치가 바뀌면 우측 하단 위치를 자동으로 다시 계산합니다.

## 친구와 함께 집중

- `방 만들기`를 누르면 Socket.IO 방 서버가 `4317` 포트에서 실행되고 초대 링크가 생성됩니다.
- 같은 네트워크의 사용자는 링크를 앱에 붙여넣거나 브라우저에서 열어 참여할 수 있습니다.
- 참여자가 집중을 시작하면 캐릭터, 닉네임, 목표, 남은 시간이 각자의 데스크톱에 별도 위젯으로 표시됩니다.
- 인터넷 외부에서도 사용하려면 `electron/room-server.cjs`를 공용 서버에 배포하고 해당 서버 주소를 사용해야 합니다.

공용 서버 실행 및 앱 연결:

```cmd
npm run room-server
set FOCUS_PET_SERVER_URL=https://your-focus-pet-server.example
npm start
```
